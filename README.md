## Featured Project

### Gemini-LMS — AI-Powered Learning Management System

A full-featured LMS with AI course generation, adaptive quizzes, and automated background workflows. Built and deployed as my final-year university project.

**Stack:** Next.js 15 · TypeScript · Drizzle ORM · Neon PostgreSQL · Google Gemini · Inngest · Clerk · PayHere · Tailwind CSS

**Scale:** 43-table relational schema, background job orchestration, secure auth, and integrated payments.

🔗 [Live demo](https://gemini-lms.vercel.app) · [Repository](https://github.com/Rashid-RG/gemini-lms)
